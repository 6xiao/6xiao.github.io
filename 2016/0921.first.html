第一篇博客

Hello, Blog!

一直以来我是个懒散的人，上完学后只勤恳了不到十年。这个月刚刚奔三成功，一些看法跟往年有很多不同的地方，就想着弄个过气的博客来记点东西。

博客的样式是从 GitHub 上直接生成后我改的，比较喜欢这种简单的感觉。

刚入职场时我曾写过两年博客但因懒停更，近年总有人给我开药方：要分享，不要独食。这跟我的认识不同：支配人的行为的更多的是习惯而不是知识，烟民难道不认识“吸烟有害健康”这几个字么？我所知道的一切，也是别人可以系统学习得来的，只要能长久的保持好奇与好学。

目前公司的情况，除我之外确实难有人能轻易弄懂我构建好的几套系统：系统非常简单却跟目前公司的整体工程水平和构建方式存在较大差异。身为把高可用、高并发、高性能、Go语言、MQ、微服务、开源这些标签贴脸上的人，无法说服自己给拉着 C++ 写 CRUD 的同事做什么普及，因为改变习惯对多数人而言真的很难。

但是最近我的观念发生了一点点变化，我觉得人与人之间更多的是共性。相似的事情总在发生，即使改变不了别人的习惯，能把我的经验和教训启发别人也是好的。

入行有些年头后，我自信积累了一些有价值的东西：关于技术、关于设计、关于架构、关于产品…… 甚至关于感情、关于家庭、关于阅读、关于投资……

如果把这些写出来后能让更多的人有所收获，那这个博客就有了它的实际意义。

2016-09-21 深圳
